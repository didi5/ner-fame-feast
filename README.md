# Döner Fame — Restaurant Product Website

> **Portfolio category:** Product & Web Applications · Business Websites  
> **Status:** Functional frontend

A responsive restaurant product website for Döner Fame at the NOVA Shoppingcenter in Leuna. The experience combines brand storytelling, menu presentation, visit information, and live opening-status logic in a modern single-page interface.

## Product highlights

- Responsive brand and menu experience
- Food and vegetarian menu sections
- Opening hours and visit information
- Live open/closed indicator using the Europe/Berlin time zone
- Local time display
- Mobile-friendly navigation and layout
- Reduced-motion accessibility handling
- Automated frontend test setup

## Technology stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui and Radix UI
- React Router
- Vitest and Testing Library

## Run locally

~~~bash
git clone https://github.com/didi5/restaurant-Fame.git
cd restaurant-Fame
npm install
npm run dev
~~~

## Quality checks

~~~bash
npm run lint
npm test
npm run build
~~~

## Current scope

This repository contains the public-facing frontend. It does not currently include online ordering, payment processing, a content-management system, table reservations, or a restaurant backend.
